== Description ==

Citadela theme is the best corner stone theme for your project. It’s fast, lightweight and super easy to work with. It can be used for any project. Small personal website, business web, portfolio website, startup, WooCommerce shop or even listing directory portal websites. It is fully responsive and full tested on all modern devices. It can be extendable by any standard 3rd party plugin such as Yoast, WPML, BuddyPress or page builders like Elementor. Code is SEO optimized and ranks great in Google search results. Citadela is developed by AitThemes - WordPress company with over 10 years of experience in creating and implementing WordPress solutions worldwide. It’s therefore secure and meets high standard.

Citadela theme can be extended with premium plugins that will shift Citadela to a whole new level. For example, you can use listing directory features on your blog or business website.
